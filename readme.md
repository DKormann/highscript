

[work in progress]

highscript is a LEAN DSP that compiles to [HVM](https://github.com/HigherOrderCo/hvm3) code

## features:

  - [x] generate HVM
  - [x] standart lambda syntax
  - [x] simple type system
  - [x] linearize variables
  - [x] run HVM output
  - [ ] Pattern matching


## installation

  - install lean

  - clone the repo

  - run `setup.sh`
